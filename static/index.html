üìã Informazioni

Il server Express ora serve contenuti statici da due cartelle:

  • public/ - File dell'applicazione (client.js, etc.)
  • static/ - Contenuti statici generici
  • static/.well-known/ - File well-known con headers specifici

üîß Configurazione

// Middleware Express configurato
app.use(express.static('public'));

// .well-known con headers specifici
app.use('/.well-known', express.static('static/.well-known', {
  setHeaders: (res, path) => {
    if (path.endsWith('apple-app-site-association')) {
      res.setHeader('Content-Type', 'application/json');
    }
  }
}));

// Altri contenuti statici
app.use(express.static('static'));
